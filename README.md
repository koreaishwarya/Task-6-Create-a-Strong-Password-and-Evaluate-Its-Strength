# Task 6 - Create a Strong Password and Evaluate Its Strength

## Objective
To understand password security principles by creating passwords of varying complexity and evaluating their strength using an online password strength checker.

## Tools Used
- PasswordMeter
- Google Chrome
- Windows 10

## Activities Performed
1. Created multiple passwords with different complexity levels.
2. Tested each password using an online password strength checker.
3. Analyzed feedback and security scores.
4. Studied common password attacks.
5. Documented password security best practices.

## Key Findings
- Longer passwords are more secure.
- Using uppercase, lowercase, numbers, and symbols improves strength.
- Weak passwords are vulnerable to dictionary and brute-force attacks.
- Unique passwords and MFA significantly enhance account security.

## Outcome
Successfully evaluated password strength and gained practical knowledge of password security and attack prevention techniques.
